# DHT11 Temperature Station

A simple Arduino-based temperature and humidity monitoring project using the DHT11 sensor and a 16x2 LCD display.

## Features

- Reads temperature and humidity from the DHT11 sensor.
- Displays the measured values on a 16x2 LCD.
- Simple and beginner-friendly Arduino implementation.

## Hardware

- Arduino Uno
- DHT11 Temperature & Humidity Sensor
- 16x2 LCD
- Jumper Wires
- Breadboard

## Software

- Arduino IDE
- KiCad (for schematic design)
